--- FULL DESCRIPTION (EN) ---

NewTabFolders transforms your new tab page into a powerful, beautiful productivity dashboard. Organize bookmarks, manage tabs, search the web, and track your most-visited sites — all without leaving your new tab.

✨ KEY FEATURES

🔍 Web Search Bar
• Built-in search bar on every new tab
• Switch between Google, Bing, DuckDuckGo, and YouTube instantly
• Remembers your preferred search engine

📊 Live Widgets
• Real-time clock with personalized greeting (Good morning / Good evening)
• Live weather widget with temperature, conditions & city name
• Powered by Open-Meteo — no API key required, 100% free

📁 Folder Management
• Create unlimited bookmark folders with custom colors and emojis
• Pin important folders to the top
• Drag & drop to reorder folders and links
• Folder templates: Blank, Work, Reading, Social, AI Tools
• Starter Packs: set up a full workspace in one click
  — Remote Worker (free), Student (free)
  — Cinema & TV, Influencer, Gamer (Pro)

⚡ Quick Access Bar
• Pin any bookmark to the Quick Bar for one-click access
• Automatically surfaced "Most Visited" section
• Click tracking — your most-used links always at the top

🔖 Smart Bookmarks
• Add links with custom icons (emoji or URL)
• Add tags to categorize and filter bookmarks
• Add private notes to any bookmark
• Duplicate link warning to avoid clutter
• Broken link checker — find dead links instantly

🔄 Tab Management
• See all open tabs in the sidebar
• Drag tabs directly into bookmark folders
• Stash all open tabs to a new folder in one click
• Import Chrome Tab Groups as folders
• Import Chrome Bookmarks in bulk (Pro)
• Close duplicate tabs automatically
• Restore recently closed tabs

🎨 Themes & Custom Backgrounds
• 5 premium themes: Dark, Light, Cyberpunk, Nord, Ocean
• 8 gradient background presets
• Set any image as your background with a custom URL
• Glass effect UI adapts to custom backgrounds

☁️ Cloud Sync (Pro)
• Sign in with Google to sync across all devices
• Real-time sync via Firebase
• Your workspace — everywhere

⌨️ Productivity
• Undo deleted folders and links (Ctrl+Z)
• Keyboard shortcuts panel
• Paste a URL anywhere to instantly add it as a bookmark
• Column count selector (2, 3, 4, or auto)
• Compact mode for more content in less space

🌍 6 Languages
English · Turkish · German · French · Portuguese · Spanish

🔒 Privacy First
• Free tier: all data stored locally on your device
• No external tracking, no analytics, no ads
• Open source — inspect every line at github.com/Methefor/New-Tab-Folders
